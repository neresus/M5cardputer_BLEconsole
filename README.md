# M5cardputer_BLEconsole
cardputer BLE console for dragino devices

Before compilation install library M5Cardputer v1.1.1 or newer and [NimBLEDevice v2.5.0](https://github.com/h2zero/NimBLE-Arduino) / Nainstaluj kniznice
Compile / Skompiluj
Export binary trough sketch / Exportni bin cez menu Sketch
Upload to SD card and load with your launcher / Nahraj na SD a instaluj cez svoj oblubeny launcher
